Create a mobile-first homepage based on the attached wireframe layout.

The page is for a digital magazine website and should preserve the structure of the sketch while translating it into a polished, production-quality UI.

## Overall Direction

- Build a clean editorial homepage with soft rounded corners and a modern magazine feel.
- Keep the composition compact and card-based, optimized first for mobile width.
- Maintain clear visual hierarchy and strong separation between sections.
- Use a simple, readable layout rather than a busy marketing page.

## Homepage Structure

### 1. Hero Wrapper

Create a large rounded container near the top of the page that acts as the hero section.

Inside this hero wrapper:

- Place a small circular logo badge in the top-left corner.
- Place a small pill-shaped `About Us` button in the top-right corner.
- Add a centered heading area labeled as the main hero content.
- On the right side of the hero wrapper, place a tall rounded vertical hero image block.
- On the lower-left side of the hero wrapper, place a featured magazine card.

### 2. Featured Magazine Card

The featured card sits inside the hero section under the hero text and to the left of the hero image.

Card content:

- magazine label or category at the top
- magazine title
- short description
- optional CTA such as `Read Now`

The card should feel like a highlighted featured issue preview.

### 3. Secondary Two-Column Section

Below the hero wrapper, create two equally tall rounded cards placed side by side:

- Left card: `Comments`
- Right card: `Contact us`

These should read as strong navigation or CTA panels.

### 4. Footer

At the bottom, place a full-width rounded footer bar spanning the content width.

## Layout Rules

- Mobile-first design, based on a narrow phone viewport.
- Use a centered page container.
- Preserve the same section order as the sketch:
  - hero section
  - two CTA blocks
  - footer
- Keep the hero section visually dominant.
- The hero image should be tall and vertically oriented.
- The featured magazine card should be smaller than the hero image but clearly emphasized.

## Visual Style

- Rounded corners throughout
- Soft, friendly, modern visual language
- Clean spacing using a consistent rhythm
- Light background with colorful section blocks
- Strong contrast for headings and labels

Suggested visual interpretation of the wireframe colors:

- Hero wrapper: light neutral or muted gray surface
- Logo badge: light blue accent
- About button: pale green accent
- Featured magazine card: warm yellow and pink editorial accents
- Comments card: strong blue block
- Contact card: mint or aqua block
- Footer: bright sky-blue block

## UX Expectations

- Make each major card feel tappable.
- Ensure tap targets are thumb-friendly.
- Use smooth hover and press feedback where appropriate.
- Keep the reading flow simple and obvious.

## Content Intent

This homepage should communicate:

- brand identity at the top
- one featured magazine issue inside the hero section
- quick navigation to engagement areas like comments and contact
- a simple footer at the bottom

## Responsive Behavior

- On mobile, keep the structure close to the wireframe.
- On tablet and desktop, scale it up cleanly while preserving the same hierarchy.
- Avoid destroying the original composition when adapting to larger screens.

## Output Goal

Turn this rough sketch into a refined homepage design prompt that can be used to build the UI in code while preserving the original layout idea.
